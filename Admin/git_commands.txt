git help config

Chapter 2:
There is 3 git directories:
working, indexed, repository

There is 4 file states in git work directory:
untracked, tracked (non-modified), modified, indexed

git add/commit/rm/clone/init/status/diff/mv/log/fetch/push/pull/reset/remote/tag
git status -s # short format
git diff # compare WORK with INDEX
git diff --staged/cached # compare INDEX with HEAD
git commit -a # add to index all modified files
git commit --amend # recommit last commit

git log -p -2 # show all diffs from last 2 commits
git log --stat # show short statisctics for every commit
git log --pretty=format:"..." --since --until --grep --author ...

git reset HEAD filename/-- # unstage file
git checkout filename/-- # discard changes in working directory

git remote -v # show all remote repos
git remote add name url # add remote repo
git remote rename nameold namenew
git remote rm
git remote show origin # show repo state

git tag # lightweight tag
git tag -a # tag with comment
